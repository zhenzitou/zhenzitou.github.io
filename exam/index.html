📖 游戏规则

🎯 游戏目标

在一学期16周内，合理分配时间完成作业、学习提升、保持身心健康，最终取得好成绩。

🏫 班级类型

  • 理科班：只需学习数学一门学科
  • 文科班：需要学习语文、英语
  • 综合版：需要学习语数英（默认选项）
  • 一带一路班：需要学习语文、数学、英语、外语

📊 属性系统

  • 学科水平：根据班级类型学习不同学科（根据难度初始总和不同，范围0-100）
  • 生理状态：范围0-50，过低会触发猝死结局
  • 心理状态：范围0-50，过低会触发自杀结局
  • 综合状态：影响行动成功率

⏰ 时间分配

每周根据难度有不同单位时间，可分配给以下行动：

  • 作业：完成可提高平时成绩，成功率 = 状态/100 × √(学科水平)/10 × 难度系数
  • 学习：提升学科水平1-5点，成功率 = 状态/200
  • 玩耍：心理状态 ±1,3,5,7
  • 睡觉：剩余时间自动睡觉，影响生理状态

💤 睡觉机制

  • 睡觉时间 ≤ 0：生理状态-20
  • 睡觉时间 ≤ 7：生理状态减少 rand(7,10)/睡觉时间
  • 睡觉时间 > 7：生理状态增加 睡觉时间-7

📝 考试机制

  • 每4周进行一次考试（第4、8、12、16周）
  • 考试成绩 = 学科水平 × 状态/100 ± 5分随机波动
  • 成绩 ≤ 40分：接下来4周总时间减少1（参加补差班）

📉 每周衰减

  • 每周生理状态随机衰减0-2
  • 每周心理状态随机衰减0-2
  • 每周学科水平随机衰减0-2（不显示）

🏆 最终成绩

  • 平时成绩（作业完成率）：占30%-70%随机权重
  • 考试成绩（4次平均分）：占剩余比例

⚠️ 结局条件

  • 生理状态 ≤ 5 → 猝死结局
  • 心理状态 ≤ 5 → 自杀结局
  • 完成16周 → 正常结局，显示最终成绩
